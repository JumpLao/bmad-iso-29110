## Change Control Process

1. **Submit** — Change request raised with description and rationale
2. **Assess** — Impact analysis on scope, schedule, cost, quality
3. **Approve/Reject** — Decision by change authority (PM / sponsor)
4. **Implement** — Approved changes assigned and executed
5. **Verify** — Implementation validated against acceptance criteria
6. **Close** — Change record completed and archived
